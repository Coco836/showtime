Show Time

Votre suivi de séries et films, en local, sans compte.

1 · Clé API TMDB

Créez un compte gratuit sur themoviedb.org, puis récupérez votre clé dans Paramètres → API (champ « Clé d'API » / API Key v3).

2 · Import TV Time (optionnel)

Chargez le fichier import-tvtime.json généré depuis votre export. Vos séries seront reliées à TMDB et vos épisodes vus restaurés.
